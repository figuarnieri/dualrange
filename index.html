A javascript library for input range with double values, with only 2kb

Star Download

DualRange.js

A javascript library, without dependencies like jQuery, Zepto and etc ... Adds double values in an input range, with a callback to facilitate implementation of Min and Max values of the input, and with a minimum support for mobile

Examples

Simple

HTML

<input type="range" />

JS

new DualRange('[type="range"]', (e) => { var min = document.querySelector('.min'); var max = document.querySelector('.max'); min.textContent = e.min; max.textContent = e.max; });
0 /// 100

Min and Max

HTML

<input type="range" min="-10" max="175" />

JS

new DualRange('[type="range"]', (e) => { var min = document.querySelector('.min'); var max = document.querySelector('.max'); min.textContent = e.min; max.textContent = e.max; });
-10 /// 175
